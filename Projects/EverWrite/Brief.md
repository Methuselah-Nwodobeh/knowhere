## Description:
A simple note taking application that backs up to GitHub, google drive, or a hosting platform.

## Aim:
Build an obsidian alternative and make it's features free. It's should be a clone of Obsidian

## Features:
- Simple note taking
- Markdown formating
- Word processing with a tool bar and shortcuts
- regular backups
- store notes as files
- read notes aloud
- convert audio to text

### User Story Map for Note Taking App

#### Epic: Note Taking
- As a user, I want to take simple notes so that I can capture my thoughts and ideas quickly.

    **User Stories:**
    - As a user, I want to create a new note.
    - As a user, I want to edit an existing note.
    - As a user, I want to delete a note.
    
#### Epic: Markdown Formatting
- As a user, I want to format my notes using Markdown so that I can structure and style my content effectively.

    **User Stories:**
    - As a user, I want to apply basic Markdown formatting (e.g., headings, lists, bold, italic).
    - As a user, I want to preview the Markdown rendering of my notes.
    
#### Epic: Word Processing
- As a user, I want to have word processing capabilities with a toolbar and shortcuts so that I can easily format and edit my notes.

    **User Stories:**
    - As a user, I want to have a toolbar with formatting options (e.g., bold, italic, underline).
    - As a user, I want to use keyboard shortcuts for common formatting actions (e.g., Ctrl+B for bold).
    
#### Epic: Regular Backups
- As a user, I want my notes to be regularly backed up so that I don't lose any important information.

    **User Stories:**
    - As a user, I want to enable automatic backups of my notes.
    - As a user, I want to specify the backup frequency and location.
    
#### Epic: File Storage
- As a user, I want my notes to be stored as files so that I can easily access and manage them outside of the app.

    **User Stories:**
    - As a user, I want each note to be saved as a separate file.
    - As a user, I want to organize my notes into folders or directories.
    
#### Epic: Read Notes Aloud
- As a user, I want the ability to have my notes read aloud to me so that I can listen to them while multitasking or on the go.

    **User Stories:**
    - As a user, I want to select a note and have it read aloud.
    - As a user, I want control over the reading speed and voice.

#### Epic: Convert Audio to Text
- As a user, I want to be able to convert audio recordings to text notes so that I can easily capture spoken ideas or conversations.

    **User Stories:**
    - As a user, I want to upload audio files and have them transcribed into text.
    - As a user, I want to review and edit the transcribed text before saving it as a note.

This user story map covers the major features and functionalities of a note-taking app like Obsidian. Each epic contains user stories that detail specific actions or requirements from the user's perspective.
